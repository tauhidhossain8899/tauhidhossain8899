## Featured Projects

### Supply Chain Inventory Analytics

**Tools Used:** SQL, MySQL  
**Project Type:** Business Analytics | Inventory Analytics | Supplier Performance Analysis

#### Project Overview
This project analyzes supply chain and inventory data to identify business problems related to stock levels, supplier performance, revenue generation, logistics efficiency, and defect rates.

#### Key Analysis Performed
- Identified low-stock and high-stock products
- Analyzed revenue by product category
- Evaluated supplier performance
- Compared shipping carriers and transportation modes
- Classified inventory status using SQL CASE statements
- Ranked products and suppliers using SQL window functions

#### SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements
- Subqueries
- RANK()
- DENSE_RANK()

#### Business Value
The analysis helps businesses improve inventory planning, reduce stockout risk, monitor supplier quality, and optimize logistics decisions.
